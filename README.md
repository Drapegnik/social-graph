## install
```
git clone https://github.com/Drapegnik/social-graph.git
git checkout release
```
* add `local.host` to your hosts file  point to `127.0.0.1`

## run
```
npm i
grunt
node server.js
```
open [local.host:3000](http://local.host:3000)
